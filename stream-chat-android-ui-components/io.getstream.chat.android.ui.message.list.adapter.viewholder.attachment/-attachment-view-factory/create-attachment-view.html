createAttachmentView

open fun createAttachmentView(data: MessageListItem.MessageItem, listeners: MessageListListenerContainer?, style: MessageListItemStyle, parent: ViewGroup): View
Content copied to clipboard

Create a content view for particular collection of attachments. If the collection contains only link attachment then it creates a link content attachment view, if the collection contains attachments without links then it creates a content view for the list of attachments, otherwise it creates a content view with both links and list contents views.

Return

View as content view for passed attachments.

Parameters

data

MessageListItem.MessageItem with particular data and attachments for the message list.

listeners

MessageListListenerContainer with listeners for the message list.

style

MessageListItemStyle style container with text colors params for the message list.

parent

View of VH's root where such attachment content view is supposed to be placed.
