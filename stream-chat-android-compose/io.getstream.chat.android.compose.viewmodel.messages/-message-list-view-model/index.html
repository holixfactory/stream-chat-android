ViewModel responsible for handling all the business logic & state for the list of messages.

Parameters

chatClient

Used to connect to the API.

chatDomain

Used to connect to the API and fetch the domain status.

channelId

The ID of the channel to load the messages for.

clipboardHandler

Used to copy data from message actions to the clipboard.

messageLimit

The limit of messages being fetched with each page od data.

enforceUniqueReactions

Enables or disables unique message reactions per user.

showDateSeparators

Enables or disables date separator items in the list.

showSystemMessages

Enables or disables system messages in the list.

dateSeparatorThresholdMillis

The threshold in millis used to generate date separator items, if enabled.
